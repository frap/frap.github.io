Bohdi Linux and Btrfs

Jul 8th, 2013

So I’m looking into docker and dockerlite for lightweight virtualisation – aka LXC’s. Docker is programmed in go and not knowing anything about go I stumbled across dockerlite. It uses bash scripts and btrfs. Btrfs is a cool disk format that allows sub-volumes as well as a lot of other nifty features. Therefore I thought I would install btrfs on my bohdi linux VM which had an ext4 disk and start checking out its reliability. I used an ubuntu live CD to reconfigure the partiton following these conversion from ext to btrfs instructions.

It seems to have worked well.
